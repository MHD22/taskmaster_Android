# Taskmaster Android App:

This App will contains tasks to do.. and you can add tasks and display your all tasks
There ara a few main activities:

1. `MainActivity`
2. `AddTaskActivity`
3. `AllTasksActivity`
4. `TaskDetailsActivity`
5. `SettingsActivity`
